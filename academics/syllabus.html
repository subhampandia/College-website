CS-101 Fall 2025
Introduction to Programming
4 Credits MWF 10:00-11:00

Instructor: Dr. Sarah Johnson

Description: Introduction to programming concepts using Python. Topics include variables, data types, control structures, functions, and basic algorithms.

Prerequisites: None

Learning Outcomes:

  • Write and debug basic Python programs
  • Apply problem-solving techniques to programming challenges
  • Understand fundamental programming concepts
BUS-201 Fall 2025
Business Management
3 Credits TTh 14:00-15:30

Instructor: Dr. Michael Chen

Description: Comprehensive study of management principles including planning, organizing, leading, and controlling in modern business environments.

Prerequisites: BUS-101

Learning Outcomes:

  • Apply management theories to real-world scenarios
  • Develop strategic planning skills
  • Understand organizational behavior and leadership
ENG-301 Fall 2025
Structural Engineering
4 Credits MWF 13:00-14:30

Instructor: Dr. Robert Martinez

Description: Advanced study of structural analysis and design principles for civil engineering applications including steel and concrete structures.

Prerequisites: ENG-201, MATH-202

Learning Outcomes:

  • Analyze complex structural systems
  • Design safe and efficient structures
  • Apply building codes and standards
CS-250 Fall 2025
Data Structures & Algorithms
3 Credits TTh 10:00-11:30

Instructor: Dr. Emily Thompson

Description: In-depth exploration of fundamental data structures and algorithmic techniques for efficient problem solving.

Prerequisites: CS-101

Learning Outcomes:

  • Implement and analyze common data structures
  • Design efficient algorithms
  • Understand complexity analysis and optimization
